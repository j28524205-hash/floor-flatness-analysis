# 바닥면 평탄성 분석 웹 앱 개발 문서

> 점군 데이터 기반 건축 마감면 품질 측정 알고리즘의 실제 구현체. 한일 심포지엄 논문(`한일심포지엄_영문초안.md`)에서 서술한 알고리즘의 소스 코드이며, 이 문서는 앱 자체의 개발/유지보수 기록이다.

---

## 1. 개요

- **목적**: 3D 점군 데이터를 입력받아 바닥면 평탄도(돌출/함몰/균열)를 자동 측정하고, 웹 UI로 즉시 시각화·보고서 출력
- **실행 방식**: 로컬 Streamlit 앱
- **주요 파일**:
  - `floor_flatness_app.py` — 앱 본체 (알고리즘 + Streamlit UI)
  - `run_flatness.py` — 실행 진입점 (`streamlit run`을 서브프로세스로 호출)
  - `바닥면_평탄성_분석_실행.bat` — 더블클릭 실행용 배치 파일
  - `floor_flatness_analysis.ipynb` — 초기 알고리즘 실험용 노트북

## 2. 실행 방법

```
바닥면_평탄성_분석_실행.bat 더블클릭
→ python run_flatness.py 실행
→ streamlit run floor_flatness_app.py (서브프로세스)
→ 브라우저에서 http://localhost:8501 자동 오픈
```

## 3. 알고리즘 파이프라인 (5단계)

| 단계 | 함수 | 설명 |
|---|---|---|
| 1 | `remove_outliers(points, k=20, std_ratio=2.0)` | SOR 이상치 제거. k-최근접 평균거리 기준 2.0σ 초과 포인트 제거 |
| 2 | `separate_floor(points, z_margin, normal_angle_thresh)` | Z값 임계 + 법선 각도(35° 미만) 필터로 바닥/벽체 분리 |
| 3 | `fit_plane_ransac(points, threshold_m, n_iter=1000, seed=42)` | RANSAC 1,000회 반복으로 기준 평면 추정 |
| 4 | `detect_cracks(points, k=15)` | 곡률(40%)+법선 불연속성(40%)+비등방성(20%) 가중 결합 균열 점수 |
| 5 | UI 내 분류 로직 | KCS 41 46 01(±3mm) + 균열 점수로 A/B1/B2/C 4등급 분류 |

- 판정 기준: 불량 비율 우수(≤5%)/양호(≤15%)/보통(≤30%)/불량(>30%)
- 시각화: 2D 편차 히트맵, 3D 표면(Z축 과장 ×1~×500), 등급별 3D 산점도, 편차 히스토그램
- 출력: 결과 CSV, TXT 요약, PDF 보고서(히트맵+히스토그램+3D 포함)
